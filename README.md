# Projeto de Projeto Programação Orientada a Objetos
## O que foi feito?
Esta é uma aplicação para gerir os serviços fornecidos por um ginásio (com aulas de grupo e individuais).

### Funcionalidades:
- A interface com o utilizador é em texto.
- A aplicação tem persistência de dados, ao utilizar objetos do tipo File e ObjectStreams para armazenar os dados.
- As classes estão organizadas com hierarquias.
- Há dados publicos como privados conforme o necessário.
- A aplicação foi documentada utilizando JavaDoc.
